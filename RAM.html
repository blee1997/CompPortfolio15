RAM

RAM stands for Random Access Memory. It is what you are actually using at any one time. Similar to your brain, RAM allows you to temporarily run applications without creating more hard drive space. This also allows for a faster method of finding information that the processor requires. Having RAM allows your computer to run faster because it provides temporary information that would be hidden deep within the hard drive.
