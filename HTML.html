HTML

HTML stands for Hyper text Mark up Language used in coding as a standardize system for tagging text files to change the font, color, appearance, and hyper link effects. This is the building blocks to creating websites because a web browser reads HTML and CSS codes to create visual and audio effects. In 1991, HTML tagging was refined by Tim Berners-Lee who was a contractor or a company called CERN.

How HTML works

HTML works with commands in a text document. To tell that you are doing a command, you would put it in <,> brackets. For example if I wanted to create a large heading, I would do then type what ever I want after the command. To end the command, you would put a forward slash (/) to tell that the command is done. The whole code would look as so, (h1) type what ever you want here(/h1) but replace the (,) with <,>

What is CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language. This is usually used to design a website. Using the Box model as a reference, many web designers can create millions of different layouts for a single website. CSS is written differently compared to html. CSS requires groups or specific ids to know which one you are trying to change. For example, you would start off a CSS code with the thing you are trying to change. Then follow up with a Bracket {,} to open it then a closed bracket at the end when you are done with the command.

Tags

Tags or commands are used to communicate with the computer to tell it what is needs to do. It is writen inbetween 2 brackets for example, < example command...> in between those brackets, you would put a certain phrase or word to initiate the command. One of the most common command is the P command or Paragraph. It is writen as < P > some working here and close it off with(/p) where you replace the (,) with <,>

Attributes

  • HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specified in the start tag
  • Attributes come in name/value pairs like: name="value"
